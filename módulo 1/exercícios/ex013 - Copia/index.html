Estilizações em CSS

Em CSS, existem 3 maneiras de se estilizar uma página, veja quais elas são!

CSS in line

É possível criar estilos para o HTML nas próprias linhas HTML usando a tag <style>. Não é uma opção recomendada pois ela torna o código muito desorganizado, além de ser uma opção não-profissional para se fazer um serviço. EVITE USAR!

CSS interno

Essa é uma forma mais organizada de criar estilos em CSS, porém, ainda não é a melhor. É recomendada quando for necessário criar estilos em apenas uma página, configuração local. Ela pode ser feita posicionando a tag <style> dentro de <head>. Ela tem 2 grandes desvantagens:

  1. Ocupa muito espaço. Sempre acaba tendo o código maior do que o HTML
  2. Se você trabalhar com mais páginas, você terá que copiar e colar a parte de estilos <style> em cada uma das páginas, o que faz perder um pouco da praticidade.
