Principais Formatações

Negrito / Destaque

- Nessa frase temos um termo em negrito usando a tag <b> (Não semântica)

- Nessa frase temos um termo em destaque usando a tag <strong> (Semântica)

Itálico / Ênfase

- Nesta frase temos um termo em itálico usando a tag <i> (Não semântica)

- Nesta frase temos um termo em destaque usando a tag <em> (Semântica)

Texto marcado

- Podemos também criar um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Texto Deletado

Texto Inserido
